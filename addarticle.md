  <p>
    Creating a custom article now runs through a GitHub issue form. Open it on
    any device where you're signed into GitHub — web or the mobile app. When you
    submit, a workflow verifies it's you, generates the post, commits it to
    <code>main</code>, then comments back and closes the issue.
  </p>
  <p><strong>No token to paste, nothing to redeploy.</strong></p>
